# Automated Retail Sales Analytics Pipeline

An automated retail sales analytics pipeline built using Python, SQL, and Pandas for data ingestion, cleaning, transformation, storage, and reporting.

---

## Features

- Automated data ingestion
- Data cleaning and preprocessing
- SQL database integration
- Sales trend analysis
- Automated report generation
- Data visualization dashboard

---

## Technologies Used

- Python
- Pandas
- SQL
- Streamlit
- Matplotlib

---

## Workflow

Raw Sales Data → Cleaning → Transformation → SQL Storage → Analytics Dashboard
