# Automated Retail Sales Analytics Pipeline

## Overview
This project is an automated retail sales analytics pipeline built using Python, Pandas, and APScheduler. The pipeline performs data cleaning, transformation, analysis, and visualization on retail sales data.

## Features
- Automated data cleaning
- Missing value and duplicate checking
- Sales analysis and revenue insights
- Data visualization using Matplotlib
- Scheduled pipeline automation using APScheduler

## Technologies Used
- Python
- Pandas
- Matplotlib
- APScheduler

## Project Workflow
Raw CSV Data
→ Data Cleaning
→ Data Transformation
→ Sales Analysis
→ Visualization
→ Automated Scheduling

## Files
- clean_data.py → Cleans and processes raw data
- analysis.py → Performs analysis and generates charts
- scheduler.py → Automates pipeline execution

## Output
- Cleaned dataset
- Revenue insights
- Category-wise sales chart
