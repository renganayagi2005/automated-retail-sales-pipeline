# Automated Retail Sales Analytics Pipeline

This project is an automated sales analytics pipeline built using Python, SQL, and Pandas to process retail sales data from raw input to report generation. The system performs data cleaning, transformation, SQL storage, and analytics reporting with scheduled execution workflows.

## Features
- Automated retail sales data processing
- Data cleaning and transformation using Pandas
- SQL database storage and querying
- Scheduled pipeline execution
- Automated analytics report generation
- Data visualization for sales insights

## Pipeline Workflow

1. Raw retail sales data is collected from CSV/API sources.
2. Python and Pandas are used for data cleaning and preprocessing.
3. Processed data is transformed and stored in SQL database tables.
4. SQL queries generate sales insights and performance metrics.
5. Automated scripts execute the workflow at scheduled intervals.
6. Final reports and visualizations are generated for business analysis.

## Tech Stack
- Python
- Pandas
- SQL
- Streamlit
- Matplotlib
- CSV / API Data Sources
